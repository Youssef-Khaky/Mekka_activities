# Web UI Kit — فعاليات مكة

Responsive Home / discovery feed. Components are JSX, loaded via Babel standalone.

## Files
- `index.html` — interactive demo (mounts `<App/>` from `App.jsx`)
- `Header.jsx` — sticky top nav with search
- `Hero.jsx` — editorial hero with type slab over photo
- `CategoryRail.jsx` — horizontal scrolling category tiles
- `FeaturedEvent.jsx` — large "happening today" feature
- `ListingGrid.jsx` + `ListingCard.jsx` — main feed of places & events
- `Footer.jsx` — footer with social, links

Real photography is missing — placeholders use warm gradient blocks.
